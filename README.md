# Random Excuse Generator 

A Python GUI-based Excuse Assistant that generates relevant excuses based on the user's problem and selected category.

## Features
- Console-based excuse generator
- GUI-based excuse generator using Tkinter
- Category-based excuses (School, Work, Personal)
- Random and relevant excuse generation

## Technologies Used
- Python
- Tkinter
- Git & GitHub

## How to Run the Project
1. Clone the repository
2. Open the project folder
3. Run the GUI file:
   ```bash
   python gui.py
